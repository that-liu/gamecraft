# 卡牌对战游戏 设计文档 (GDD)

## 1. 游戏概述
**游戏名称**: 暂定"维度之战"(Dimensional Wars)  
**类型**: 策略卡牌对战  
**平台**: PC / 手游  
**目标用户**: 16-35岁，喜欢策略游戏的玩家  

## 2. 核心玩法
- 玩家收集卡牌，组建30张卡的卡组
- 每回合获得1点法力值（上限10点）
- 使用法力值召唤怪物、施放法术、设置陷阱
- 击败对方玩家（生命值归零）即获胜

## 3. 卡牌系统

### 怪物卡
| 属性 | 说明 |
|------|------|
| 攻击力 | 1-12 |
| 防御力 | 0-10 |
| 生命值 | 1-15 |
| 法力消耗 | 1-10 |
| 特殊技能 | 战吼、亡语、光环、冲锋、嘲讽 |

### 法术卡
- 直接伤害（火球术：3费，造成5点伤害）
- 增益效果（力量祝福：2费，+3攻击力）
- 减益效果（虚弱诅咒：1费，-2攻击力）
- 抽卡效果（智慧启迪：3费，抽3张牌）

### 陷阱卡
- 反击陷阱：敌方攻击时触发，反弹伤害
- 召唤陷阱：敌方出牌时触发，召唤一个1/1生物
- 破坏陷阱：敌方出牌时触发，摧毁该卡

## 4. 对战系统
- 回合制，每人每回合60秒
- 法力水晶每回合+1（最多10）
- 手牌上限10张
- 场上怪物上限7个

## 5. 游戏模式

### PVE 剧情模式
- 8个章节，每章5关
- Boss战有特殊规则
- 通关解锁新卡牌
- 难度递增AI

### PVP 对战模式
- 天梯排名（青铜→白银→黄金→钻石→传说）
- 休闲匹配
- 好友对战
- 赛季重置（每月）

## 6. 经济系统
- 金币：对战/任务获得
- 宝石：付费/稀有奖励
- 卡包：5张随机卡（1张稀有保底）
- 战令：68元/月，解锁额外奖励

## 7. 技术方案
- 前端：Unity 2D
- 后端：Go + WebSocket
- 数据库：PostgreSQL
- 匹配服务：Redis
